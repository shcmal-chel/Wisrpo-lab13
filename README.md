## Описание проекта

Данный проект создан в рамках лабораторной работы по изучению CSS-фреймворка Bootstrap 5.

## Что было изучено:
* Подключение Bootstrap через CDN
* Система сеток (Grid System)

* Утилитные классы
* Адаптивные изображения
* Валидация форм
* Кастомизация стилей
